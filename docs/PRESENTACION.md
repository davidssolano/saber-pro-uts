# Guion para sustentación
1. **Contexto Saber Pro**: importancia para acreditación y seguimiento del programa de Ingeniería de Sistemas.
2. **Problema**: falta de trazabilidad de resultados y beneficios; uso de Excel dispersos.
3. **Objetivo**: sistema web centralizado para resultados, incentivos y reportes.
4. **Requerimientos**: listar funcionales y no funcionales (seguridad, despliegue en Render, backups).
5. **Historias de usuario y Scrum**: backlog priorizado y sprints.
6. **Arquitectura**: Java Spring Boot + Thymeleaf + MySQL/TiDB, despliegue en Render; capas controller/service/repository.
7. **Modelo de datos**: entidades Estudiante, Resultado, Incentivo y relación para beneficios.
8. **Demostración**: CRUD estudiantes, resultados, incentivos y reportes.
9. **Pruebas**: casos clave ejecutados y pendientes.
10. **Conclusiones y trabajo futuro**: integración de login/roles, exportaciones, auditoría y analítica avanzada.
